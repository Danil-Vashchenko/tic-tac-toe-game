# Tic Tac Toe Game

Цей проєкт створено з метою ознайомлення з основами мови програмування Python та створення простої гри за допомогою модуля `turtle`.

Основною метою проєкту було розібратися зі створенням логіки гри “Хрестики-нулики”, навчитися обробляти натискання миші, перевіряти перемогу гравця, нічию та створити простого бота для гри проти користувача.

## Основні модулі проєкту

`turtle` - Основний модуль для створення графічного інтерфейсу гри в Python.

## Структура проєкту

`tic_tac_toe.py` - Головний файл запуску проєкту, у якому створено ігрове поле, логіку ходів, перевірку перемоги та роботу бота.

## Опис проєкту

У грі створено поле 3x3 для гри “Хрестики-нулики”.

Гравець грає за хрестики, а бот грає за нулики.
Гравець робить хід за допомогою натискання миші по клітинці ігрового поля.

Після ходу гравця бот автоматично аналізує поле та робить свій хід.

## Основна логіка гри

У проєкті реалізовано:

* створення ігрового поля;
* малювання хрестиків;
* малювання нуликів;
* обробку кліків миші;
* перевірку переможця;
* перевірку нічиєї;
* виведення результату гри на екран;
* просту логіку бота.

## Логіка бота

Бот працює за простим алгоритмом:

1. Спочатку бот перевіряє, чи може він перемогти своїм ходом.
2. Якщо перемогти не може, бот перевіряє, чи може гравець перемогти наступним ходом.
3. Якщо гравець може перемогти, бот блокує цей хід.
4. Якщо небезпеки немає, бот обирає першу вільну клітинку.

## Запуск проєкту

Для запуску проєкту необхідно виконати такі кроки:

### Встановлення ресурсів

Python - Головна мова програмування. Якщо вона не встановлена, її потрібно встановити.

Git - Основний ресурс для контролю версій проєкту, на якому його збережено.

### Встановлення та підготовка проєкту

Встановлення проєкту - в терміналі виконайте команду:

```bash
git clone https://github.com/Danil-Vashchenko/tic-tac-toe-game.git
```

Перейдіть у папку проєкту:

```bash
cd tic-tac-toe-game
```

Для запуску проєкту запустіть файл `tic_tac_toe.py`.

Або в терміналі напишіть команду:

```bash
python tic_tac_toe.py
```
